Using Absolute or Relative Units

Article: The Surprising Truth About Pixels and Accessibility
Rating: 4.5/5
Description

Josh Comeau talks about when to use pixels (absolute units) and when to use rem (relative uints). Pixels are great for things that we always want to be the same size. Using relative units is great when we want things to be fluid. There are times to use either, we shouldn't restrict ourselves to only one.

On Typography

Article: Typography
Rating: 4/5
Description

This is a bit more technical, but gives tips on how text should look. It talks about sizing for desktops and mobile phones. It even gives good types on tyopgraphy in general, and recommendations that even book publishers have used for hundreds of years.
