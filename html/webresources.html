Using Absolute or Relative Units

Rating: 4.5/5
Description

Josh Comeau talks about when to use pixels (absolute units) and when to use rem (relative uints). Pixels are great for things that we always want to be the same size. Using relative units is great when we want things to be fluid. There are times to use either, we shouldn't restrict ourselves to only one.
